박스를 통해 h1 태그는 화면 전체를 쓰고 a 태그는 자기 콘텐츠만큼 부피를 쓴다는 것을 알 수 있다
화면 전체를 쓰는 태그들이 몇 십 개가 있고, 자기 크기만큼 쓰는 것도 몇 십개가 있다.
화면 전체를 쓰는 태그들을 block level element이라고 한다.
자신의 부피만큼만 쓰는 태그는 inline element
하지만 블럭레벨태그라도 인라인으로 바꿀 수 있다. display 속성을 통해서.

콘텐츠와 테두리 사이에 여백을 주고 싶다면 패딩값 주기
css 박스와 css 박스 2 사이에 간격이 생기는게 싫다면 마진 0으로 주기, 혹은 원하는 값 주기
화면 전체를 쓰는게 싫다면 블럭박스의 너비 값을 지정하면 된다 display 속성을 통해서.
